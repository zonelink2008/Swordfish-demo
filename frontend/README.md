# ⚔️ Swordfish Demo

日本AI电力交易代币化平台 - 前端演示

## 功能特性

- 📊 **仪表盘** - 实时价格预测、市场统计
- 💱 **交易** - 买入/卖出电力
- 💼 **资产** - 投资组合管理
- 📈 **市场** - JEPX行情数据
- ⚙️ **设置** - KYC认证、预警配置

## 技术栈

- React 18 + Vite
- Tailwind CSS
- Recharts 数据可视化
- Ethers.js 钱包连接
- React Router 多页面

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build
```
